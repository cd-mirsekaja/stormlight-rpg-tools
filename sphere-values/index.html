Sphere Value Calculator

The Stormlight RPG uses Diamond Marks as a unified currency to make the life of players and GMs easier. There are, however, 30 different types of spheres in circulation on Roshar.
This tool can convert a number of Diamond Marks into a semi-random combination of different spheres.

How it works: Enter a value and optionally set minimum/maximum sphere types. The algorithm generates random combinations that add up exactly.
