# stormlight-rpg-tools
A suite (currently only one tool) of useful GM tools for Brandon Sandersons Stormlight Archive RPG. This repository only holds the code, the site is hosted via GitHub Pages.

## Sphere Value Calculator
A simple tool to convert any number of Diamond Marks into a corresponding number of different spheres. Useful for generating loot tables and such.
